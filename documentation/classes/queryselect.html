Class QuerySelect<T>

class

QuerySelect is private part of the APi. For design reasons this class should not be used directly and will move later. Use this class with Select function.

example
// select todos
Select(Todo).where({isDone: false}}).exec().subscribe((qr: QueryResult<Todo>) => {
     // do something with the result...
}, (err) => {
     // do something with the error...
});
