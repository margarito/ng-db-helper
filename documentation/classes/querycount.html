Class QueryCount<T>

class

QueryCount

description

For design reasons this class should not be used directly. Prefer use Count function.

example
// count todos
Count(Select(Todo).where({isDone: false}})).exec().subscribe((cout: number) => {
     // do something with the result...
}, (err) => {
     // do something with the error...
});
